Gracias por descargar Encuestame Installer ${application.version}.

Este Instalador viene incorporado con un servidor web Tomcat 7.0.59 y una Base de Datos hSQL, para que sea ejecutado fuera de la caja.

  • Establezca su directorio principal para Encuestame. Los paquetes necesarios se instalaran en un directorio de su eleccion.
  • Es recomendado que el directorio seleccionado de Instalacion tenga los permisos necesarios de lectura, escritura y ejecucion.
  • Comprobar Conexion de Base de Datos
  • Personalizar configuracion de propiedades de reCaptcha . Instrucciones de como personalizar la configuracion por defecto, pueden ser encontradasen este enlace
